实验使用Jupyter Notebook实现

in.txt和out.txt分别为输入文件和输出文件
in.txt存放待识别的token序列
out.txt存放识别结果

其中in.txt前半部分为堆排序代码
后半部分为一些特殊的合法的和非法的测试用例

种别码为-1，为数字超长。
种别码为-2，为标识符以数字开头错误。
种别码为-3，为标识符超长。
种别码为-4，为无法确定的特殊符号。
种别码为1。关键字："begin", "call", "const", "do", "end", "if", "odd", "procedure", "read",
                              "then", "var", "while", "write", "for", "printf", "return", "void", "int"
种别码为2。分界符：",", ".", ";", "(", ")", "[", "]", "{", "}", "\\", "\"", "\'", "#", ":"
种别码为3。算数运算符："+", "-", "/", "++", "--", "%", "*", "&"
种别码为4。关系运算符："<=", "<", ">=", "=", ">", ">=", "<>", ":=", "#", "!", "=="
种别码为5。常数，以0-9组成小于14位的数字
种别码为6。标识符：用来表示各种变量名，必须以字母开头小于10位字符组成
种别码为7。赋值运算符："=", "-=", "-=", "*=", "/=", "%=", "&=", "^=", "|="
种别码为8。位运算符："|", "^", "~", "<<", ">>"
